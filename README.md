# Content Generation Platform

Multilingual content generation platform for marketing copy, blog posts, and social media.

## Features
- 12 language support
- SEO optimization
- Brand voice consistency
- A/B variant generation
- Batch processing

## Installation
```
pip install -r requirements.txt
```

## Usage
```
python main.py generate --type blog --topic 'AI trends' --lang en
python main.py batch topics.csv
```
